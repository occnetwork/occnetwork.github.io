@lume I am a little late here, but this just happened to me and I found a way to restore it: Go to your local data folder located in "\AppData\Local\Pal\Saved\SaveGames", there you'll find "LocalData.sav" You'll also find a folder called "backup" and inside it a folder called "local", go into that folder. You'll find a lot of backups of the "LocalData.sav" file, each inside a folder named with date and time. Go to the folder with the date and time before your map was deleted, then copy the "LocalData.sav". Go back to the folder in "\AppData\Local\Pal\Saved\SaveGames" and paste the file there, replace the "LocalData.sav". Just make sure the game isn't running while you do this and once you go back in, the map should be restored as it was. If not, try restoring an older backup of said file.
